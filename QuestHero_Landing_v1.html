1
🗂️ 空间 + File
建空间, 再建第一个 File
空间 = 一个分类 (工作 / 生活 / 学习). 一个 file = 一件要持续推进的事 (跳槽 / 装修 / 学日语). 左上 ▾ 切换空间, 后面 [+ 新建 file].
  • 不同空间数据完全隔离
  • 建 file 时可选模板 (LO / 求职 / 装修...) 或空白
  • 一个空间下可以有任意多个 file
